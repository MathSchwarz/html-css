Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>. A tag <b> é uma tag Sem semântica.

Nesta frase, temos um termo em destaque usando a tag <strong>, que é uma tag semântica.


Itálico/ Ênfase

Nessa frase temos um termo em itálico usando a tag <i>. (Não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em>, que é uma tag semântica.


Texto Marcado

Podemos criar um texto marcado usando a tag <mark>


Tamanho de textos

Estamos criando um texto grande e um texto pequeno nesse paragrafo.


Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.


Texto Inserido

Podemos marcar um texto como inserido para dar enfase nele e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u>


Texto sobrescrito

Para inserir coisas do tipo x20+3. Sobrescrito são itens que ficam acima.


Texto Subscrito

Para inserir coisas do tipo h2o. Subscrito são itens que ficam abaixo.
