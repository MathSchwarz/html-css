Para importar fontes das quais n√£o existem no google fonts, podemos baixar a fonte no site: https://www.dafont.com/pt/ e importar no nosso site utilizando o comando @font-face {} dentro da tag STYLE.

Tipos de format ()

  1. opentype (otf)
  2. truetype (ttf)
  3. embedded-opentype
  4. ruetype-aat (Apple Advanced Typography)
  5. svg

Para que a fonte seja validada, precisamos colocar um nome para a mesma, e sua URL corretamente (conforme o nome que aparece na pasta) ex: 

            
                <style>
                @font-face {
                    font-family: 'Old';
                    src: url('fonts/OldLondon.ttf') format(truetype), url('fonts/OldLondonAlternate.ttf') format('truetype'), url('fonts/Olondon_.otf') format('opentype'), url('fonts/Olondona.otf') format('opentype');
                    font-weight: normal;
                    font-style: normal;
                    }
                    </style>
