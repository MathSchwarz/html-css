Trabalhando com Fontes

Existem dois tipos de medidas, as medidas ABSOLUTAS e as RELATIVAS. As medidas ABSOLUTAS são: cm, mm, in, px, pt, pc 16 px = 1em As medidas RELATIVAS são: em, ex, rem, vw, vh, %

Quando for usar uma medida ABSOLUTA, utilizar PX e quando for usar uma medida RELATIVA, usar o EM

Comandos

Temos uma opção de comandos chamado SHORTHAND, onde podemos reduzir o tamanho de uma personalisação, ex:

Podemos sair de: 

            
                font-family: 'Work Sans', sans-serif;
                font-weight: bolder;
                font-size: 3em;
                font-style: italic;

Para: 

            
                font: italic bolder 3em 'Work Sans', sans-serif;
Isso utilizando a SHORTHAND
