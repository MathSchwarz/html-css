Aprendendo CSS

CSS Básico

Podemos criar um seletor especifico de CSS para uma tag utilizando o código: id = "".
Supondo que demos o nome de id:"principal", dentro do CSS podemos colocar por exemplo: h1#principal ou apenas #principal, assim podemos personalizar especficamente aquela tag em questão.

O código ID ="" serve para ser usado uma unica vez, ja o código: class="" pode ser usado multiplas vezes.

Também é possivel utilizar um ID e uma classe na mesma TAG, exemplo: id="principal" class="destaque"
Mas o ID sempre sobrepõe as configurações de classe.
