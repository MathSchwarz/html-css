Testando Gradientes

Para inserirmos um Degradê no nosso site, dentro do body, podemos usar a tag: background-image: linear-gradient ai informamos a direção e as cores que desejamos ter no gradiente, ex: (to right, white, black) Também podemos utilizar a unidade "deg" quando precisamos direcionar um gradiente. Em CSS, a unidade "deg" (graus) é usada principalmente para especificar ângulos, como em transformações e em gradientes. Ex:

1. **Transformações**: Podemos pode usar graus para rotacionar elementos. Por exemplo: 

                
                    .rotate {
                           transform: rotate(45deg); /* Rotaciona o elemento em 45 graus */
                       }

2. **Gradientes**: Ao criar gradientes, podemos usar graus para definir a direção da transição de cores. Por exemplo: 

                
                    .gradient {
                           background: linear-gradient(90deg, red, blue); /* Gradiente do vermelho ao azul na direção horizontal */
                       }

3. **Sombras**: Em algumas propriedades, como box-shadow , você pode usar graus para definir a direção da sombra. Os ângulos são medidos em relação ao eixo x, onde 0deg é para a direita, 90deg é para cima, 180deg é para a esquerda e 270deg é para baixo.
