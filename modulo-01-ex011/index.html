Reproduzindo Áudio

Podemos reproduzir audios em HTML 5, utilizando as tags <audio src="" controls autoplay></audio>

Também temos outra opção de comando

            
                <audio preload="metadata" controls loop>
                <source src="./media/drums-of-libertation.mp3" type="audio/mpeg">
                <source src="./media/drums-of-libertation.mp3" type="audio/ogg">
                <source src="./media/drums-of-libertation.mp3" type="image/wav">
                </audio>

Essa opção vai tentar reproduzir em dicersos tipos de mídia. Logo após a opção control, pode se colocar o comando autoplay, para que ele reproduza automaticamente.
